# Order Management App / Application de gestion de commandes

## English

Simple application to manage store orders. Allows CRUD operations on products and orders (e.g., adding chairs, tables, quantities, total price).

### Features
- Add, update, and delete products.
- Add, update, and delete orders.
- Simple web interface using Thymeleaf.

### Technologies
- Java
- Spring Boot
- Spring Data / Hibernate
- Thymeleaf
- MySQL

### Setup
1. Configure your MySQL database in `application.properties`.
2. Open the project in your IDE.
3. Run the main class `Lanceur.java`.
4. Access the app at `localhost:8080`.


## Français

Application simple pour gérer les commandes d'un magasin. Permet de créer, modifier et supprimer des produits et des commandes (ex : ajouter des chaises, tables, quantités, prix total).

### Fonctionnalités
- Ajouter, modifier et supprimer des produits.
- Ajouter, modifier et supprimer des commandes.
- Interface web simple avec Thymeleaf.

### Technologies
- Java
- Spring Boot
- Spring Data / Hibernate
- Thymeleaf
- MySQL

### Installation
1. Configurez votre base de données MySQL dans `application.properties`.
2. Ouvrez le projet dans votre IDE.
3. Lancez la classe principale `Lanceur.java`.
4. Accédez à l'application sur `localhost:8080`.
